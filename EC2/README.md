# ℹ️ 아마존 웹 서비스(AWS)에서 제공하는 클라우드 컴퓨팅 서비스이다
#    ㄴ> 인터넷(클라우드)을 통해 서버, 스토리지, 데이터베이스 등의 컴퓨팅 서비스를 제공 → AWS에서 원격으로 제어할 수 있는 가상의 컴퓨터를 한 대 빌리는 개념
