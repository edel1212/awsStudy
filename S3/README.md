# S3( Simple Storage Service) 
```properties
# ℹ️ 아마존 웹 서비스(AWS)가 제공하는 클라우드 스토리지 서비스이다.
#   - 파일, 데이터 및 다양한 유형의 미디어 등을 저장하고 관리하는 웹 기반 스토리지 시스템이다.
```

## S3 특징
- `객체` 스토리지 서비스이다.
  - 파일만 보관이 가능하다
    -  반대 개념 : Block Storage Service(EBS, EFS 등)
    -  어플리케이션 설치 불가능
-  글로벌 서비스이다.
  - 단! 데이터는 내가 지정한 리전에만 저장된다.
- 무제한 용량
